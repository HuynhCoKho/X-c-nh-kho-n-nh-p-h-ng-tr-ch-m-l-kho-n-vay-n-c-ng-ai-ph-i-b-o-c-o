# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to the repository owner instead of opening a public issue. Include the affected page/repository, steps to reproduce, impact, and any relevant screenshots or logs.

Do not use this repository's public issue tracker to disclose secrets, vulnerabilities, private business logic, or exploit details.
